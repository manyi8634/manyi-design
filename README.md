<div align="center">

一套企业级 UI 设计语言和 React 组件库。

## 使用文档
> https://manyi8634.github.io/manyi-design-doc

## 📦 安装

```bash
npm install manyid --save
```

```bash
yarn add manyid
```

## 🔨 示例

```jsx
import React from 'react';
import { Button, DatePicker } from 'manyid';

const App = () => (
  <>
    <Button type="primary">PRESS ME</Button>
    <DatePicker />
  </>
);
```
